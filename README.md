# Trendora - E-Commerce Shopping Website

Trendora is a modern and responsive E-Commerce Shopping Website developed using HTML, CSS, and JavaScript. The application provides users with an interactive online shopping experience, including product browsing, search functionality, wishlist management, and cart operations.

## Features

- Responsive Design
- Product Listing
- Product Search
- Shopping Cart
- Wishlist Functionality
- Product Details
- User-Friendly Interface
- Dynamic Product Rendering
- Local Storage Support

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Font Awesome
- Local Storage API

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/rojamanikone1013-tech/Trendora-.git
```

2. Navigate to the project folder:

```bash
cd Trendora-
```

3. Open `index.html` in your browser.
